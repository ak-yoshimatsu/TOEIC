# TOEIC 学習リポジトリ

個人のTOEIC学習教材・問題・記録を管理するリポジトリです。

## 目標

| 試験 | 時期 | 目標スコア |
| --- | --- | --- |
| 第1回 | 2026年6月 | 500点（L335 / R165） |
| 第2回 | 2026年8月 | 700点（L400 / R300） |

## ディレクトリ構成

```md
TOEIC/
├── articles/          # 学習教材（読み物・語彙リストなど）
├── questions/
│   ├── Part5/         # 短文穴埋め問題
│   └── Part7/         # 長文読解問題
└── answers/
    ├── Part5/         # Part5の回答・解説
    └── Part7/         # Part7の回答・解説
```

## スラッシュコマンド

| コマンド | 説明 |
| --- | --- |
| `/question` | Part 5 の問題を1問出題し、`questions/Part5/` に保存する |
| `/answer [ファイル]` | 指定した問題ファイルを採点・解説し、`answers/Part5/` に保存する。引数省略時はアクティブなファイルを対象とする |
