## 解説

`ask + 人 + to do` の構文で、空欄には **動詞の原形** が入ります。

`The manager asked all employees to -------` の `to` は不定詞の `to` であるため、直後には動詞の原形が必要です。

| 選択肢 | 品詞/意味 | 判定 |
| --- | --- | --- |
| (A) submit | 動詞（原形）「提出する」 | ✓ |
| (B) submission | 名詞「提出」 | × |
| (C) submitted | 動詞（過去形・過去分詞）| × |
| (D) submitting | 動詞（現在分詞・動名詞）| × |

**ポイント**: `ask + 人 + to + 動詞の原形` はTOEICで頻出の構文。`to` の後は必ず動詞の原形（原形不定詞）になる。品詞を問う問題では文の構造を見極めることが重要。
