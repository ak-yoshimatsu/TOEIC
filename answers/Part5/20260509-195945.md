## 正解: (A) submit

## 解説

「経費報告書を月末までに〇〇するよう求めた」という文脈では、**submit**（提出する）が正解です。`submit a report / document` はビジネス英語で非常に頻出のコロケーションです。

| 選択肢 | 意味 | 判定 |
| --- | --- | --- |
| (A) submit | 「提出する」— submit expense reports で「経費報告書を提出する」 | ✓ |
| (B) suggest | 「提案する」— suggest expense reports とはつながらない | × |
| (C) support | 「支持する・補助する」— 文脈に合わない | × |
| (D) substitute | 「代替する・代わりに使う」— 文脈に合わない | × |

**ポイント**: TOEIC頻出コロケーション「動詞 + 書類・報告」の組み合わせを覚えよう。

- `submit` a report / application / form（報告書・申請書・書類を提出する）
- `approve` a report / request（承認する）
- `review` a document（見直す）

s で始まる似た単語（submit / suggest / support / substitute）は意味が全く異なるので要注意。
