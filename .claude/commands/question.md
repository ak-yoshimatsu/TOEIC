# TOEIC Part 5 出題スキル

TOEIC Part 5形式の問題を1問作成して出題する。

## 手順

1. 現在の日時を `date +"%Y%m%d-%H%M%S"` で取得する
2. 以下のテーマからランダムに1つ選んで問題を作成する
   - 品詞（名詞・動詞・形容詞・副詞の区別）
   - 時制（現在完了・過去・未来など）
   - 前置詞（in / on / at / for / since など）
   - 接続詞・接続副詞
   - 代名詞・関係代名詞
   - 語彙（TOEIC頻出ビジネス単語）
3. 問題文をTOEIC Part 5形式（4択）で作成する
4. 問題ファイルを `questions/Part5/<yyyymmdd-HHmmss>.md` に保存する
5. 問題ファイルの末尾に `**回答**: ( )` を含める
6. 問題をユーザーに提示して回答を促す

## 問題ファイルのフォーマット

```markdown
# Part 5 - Question

**テーマ**: <テーマ名>

---

<問題文。空欄は ------- で表す>

(A) <選択肢>
(B) <選択肢>
(C) <選択肢>
(D) <選択肢>

---

**回答**: ( )
```

## 注意事項

- 問題のレベルはTOEIC 500点前後を目安にする
- ビジネスシーンを題材にした自然な英文を使う
- 解説は回答後に提示する（問題と同時には見せない）
- 回答を受け取ったら `answers/Part5/<同じタイムスタンプ>.md` に解説を保存する
