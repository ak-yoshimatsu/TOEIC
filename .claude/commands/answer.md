# TOEIC Part 5 採点スキル

指定された問題ファイルの回答を採点し、評価・解説を行う。

## 引数

- 引数あり: 指定されたファイルパスを採点対象とする
- 引数なし: IDEで現在アクティブなファイルを採点対象とする

## 手順

1. 採点対象ファイルを読み込む
2. `**回答**: (X)` の記載を確認する
   - 未回答（空欄）の場合は「回答が記入されていません」と伝えて終了
3. 問題文・選択肢・回答を分析し、正誤を判定する
4. 以下の内容を含む解説を作成する
   - 正解の選択肢
   - 正誤の判定（正解 / 不正解）
   - 解説（なぜその選択肢が正解か、他の選択肢がなぜ不正解か）
   - 間違えた場合は学習ポイントをまとめる
5. 問題ファイルのタイムスタンプを取得し、`answers/Part5/<同じタイムスタンプ>.md` に解説を保存する
6. すでに解説ファイルが存在する場合は上書きする

## 解説ファイルのフォーマット

```markdown
# Part 5 - Answer

**問題ファイル**: questions/Part5/<ファイル名>
**テーマ**: <テーマ名>

---

## 正解: (X) <正解の語>

## 解説

<正解の理由を日本語で説明>

| 選択肢 | 品詞/意味 | 判定 |
| --- | --- | --- |
| (A) ... | ... | × |
| (B) ... | ... | × |
| (C) ... | ... | ✓ |
| (D) ... | ... | × |

**ポイント**: <学習者が覚えるべき文法・語彙のポイント>

## 学習者の回答

- 回答: **(X) <回答した語>**
- 結果: **正解** または **不正解**
- 間違いのパターン: <不正解の場合のみ記載>
```

## 注意事項

- 解説はすべて日本語で行う
- 間違えた問題のパターンは記録し、次回の出題に活かす
- 正解の場合も簡潔な解説を提供する
